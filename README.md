# One Line English

One Line English is a minimal iPhone app for learning English vocabulary from one meaningful line a day.

The app includes:

- A daily quote-based vocabulary card
- One highlighted advanced keyword per line
- Review for learned quotes and words
- English and Chinese app interface settings
- Local-only progress storage

The app does not require an account and does not use a backend.
